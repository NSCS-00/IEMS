## 模型格式说明

### PMX/PMD 格式 (MMD 模型)

**适用**: 中继器、广播塔等复杂模型

**坐标系统**:
```
原点：方块中心
X 轴：东正西负
Y 轴：上正下负
Z 轴：南正北负
```

**推荐设置**:
- 单位：1 像素 = 0.0625 (1/16) 方块
- 中心偏移：(0.5, 0, 0.5) 方块
- 旋转中心：方块底部中心

### JSON 方块模型格式

**适用**: 所有方块

**示例结构**:
```json
{
  "parent": "block/block",
  "textures": {
    "particle": "iems:block/energy_relay",
    "all": "iems:block/energy_relay"
  },
  "elements": [
    {
      "from": [4, 0, 4],
      "to": [12, 16, 12],
      "faces": {
        "down": {"texture": "#all"},
        "up": {"texture": "#all"},
        "north": {"texture": "#all"},
        "south": {"texture": "#all"},
        "west": {"texture": "#all"},
        "east": {"texture": "#all"}
      }
    }
  ]
}
```

## 临时方案

在模型文件提供之前，使用以下临时方案：

### 1. JSON 方块模型 (基础几何体)

使用中继器和广播塔使用简单的柱状模型：
- 中继器：4x16x4 像素柱体
- 广播塔：3x20x3 像素柱体 + 顶部小球

使用单色 PNG 贴图：
- 中继器：灰色 (#888888)
- 广播塔：白色 (#CCCCCC)
- 转换器：蓝色 (#4444FF)
- 存储器：绿色 (#44FF44)

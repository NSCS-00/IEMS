## 能源系统架构

```
                    ┌─────────────┐
                    │   [核心]     │ ← 世界电力系统的中心
                    │  (其他 MOD)  │   管理整个世界的能源
                    └──────┬──────┘
                           │
         ┌─────────────────┼─────────────────┐
         │                 │                 │
    ┌────┴────┐      ┌────┴────┐      ┌────┴────┐
    │ 中继器   │      │ 广播塔   │      │ 存储器   │
    │(标记连接)│      │(标记连接)│      │(储能备用)│
    └────┬────┘      └────┬────┘      └─────────┘
         │                 │
    其他 MOD 设备       其他 MOD 设备
```

**供电逻辑**:
1. 核心直接供电给所有连接的设备
2. 核心优先使用自身存储的能量
3. 核心能量不足时，向存储器请求能源 (以耗电功率×2)
4. 核心能量有富余时，向存储器充能
5. 中继器/广播塔本身没有能量缓存，仅作为连接标记

## Web 界面

### 文件结构

```
web/
└── WebServer.java              # HTTP 服务器实现
    ├── 主页面：http://localhost:8080/
    └── API: http://localhost:8080/api/status
```

### 特点

- 无需额外贴图文件
- 使用内嵌 HTTP 服务器
- HTML/CSS/JS 动态生成
- 每 5 秒自动刷新

## API 使用

其他 MOD 可以通过以下方式使用 IEMS:

1. **注册核心方块到电网** - 使用 `GridCoreRegistry`
2. **注册设备到广播塔** - 使用 `EnergyConnectionManager`
3. **使用 IEMS 能源单位** - 使用 `EnergyValue` 类
4. **获取电网信息** - 使用 `EnergyGrid` 单例或 Web API

详见：`API 使用示例.md`

## 核心类说明

### IEMSMod.java
主模组类，负责：
- 初始化所有系统
- 注册事件监听器
- 启动 Web 服务器

### EnergyValue.java
高精度能量值类，支持：
- SE/GE/FE/AE 单位
- BigInteger 存储
- 单位换算和显示

### EnergyGrid.java
电网系统单例，管理：
- 核心位置
- 设备连接
- 能量分配

### GridCoreRegistry.java
核心注册表，实现：
- 核心提供者注册
- 核心唯一性检查
- 多 MOD 冲突处理

## 方块系统

### 方块列表

| 方块 | 功能 |
|------|------|
| EnergyStorageBlock | 能量存储（标准/通用） |
| EnergyConverterBlock | 能源单位转换 |
| EnergyRelayBlock | 长距离连接（500 格） |
| EnergyBroadcastTowerBlock | 短距离无线传输（50 格） |
| EnergyMarkerBlock | 标记方块（不可见） |

### 方块实体

每个方块都有对应的 BlockEntity 类，处理：
- 能量存储和传输
- 连接管理
- 用户交互

## Web 系统

### WebServer.java
HTTP 服务器，提供：
- 静态 HTML 页面
- REST API 接口
- 设备管理功能

### API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | HTML 页面 |
| `/api/energy` | GET | 能量数据 |
| `/api/power` | GET | 功率数据 |
| `/api/protocol` | GET | 协议容量 |
| `/api/devices` | GET/POST | 设备列表/切换 |
| `/api/config` | GET | 配置信息 |

## 工具类

### SightCheckUtil.java
视线检查工具，实现：
- 射线追踪算法
- 方块遮挡检测
- 设备方块穿透

## 资源文件

### neoforge.mods.toml
模组元数据配置：
- Mod ID
- 版本信息
- 依赖关系

### 语言文件
- `zh_cn.json` - 简体中文
- `en_us.json` - 英文

### zcs-terminal.html
Web 监控界面：
- 4 套配色方案
- 实时数据刷新
- 设备管理功能

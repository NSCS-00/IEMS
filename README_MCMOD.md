## 📖 模组简介

综合能源管理系统是一个为大型整合包设计的高精度能源管理模组。它提供了全新的能源单位和管理体系，让你能够建造规模宏大的电力网络！

### ✨ 核心特色

- 🔋 **超高精度能源** - SE/GE 单位，支持超大数值存储
- 🌐 **Web 监控界面** - 浏览器实时查看电网状态，支持设备管理
- 🏗️ **灵活电网架构** - 中继器/广播塔构建无线能源网络
- ⚡ **核心唯一性** - 每个世界只有一个能源核心，由其他 MOD 提供

## 🎮 玩法指南

### 新手入门

#### 1. 理解能源单位

本模组使用全新的能源单位系统：

```
1 SE (标准能量) = 100,000,000 GE (通用能量)
1 GE = 9,000,000,000,000,000,000 FE (传统 Forge 能量)
```

**简单来说**：
- **FE** - 传统模组使用的能量单位
- **GE** - 本模组的中等单位
- **SE** - 本模组的高级单位，用于大型电网

#### 2. 核心工作原理

```
┌─────────────────────────────────────┐
│          核心 (由其他 MOD 提供)        │
│                                     │
│    ↓ 供电 ↓ 充电 ↓ 平衡能源          │
│                                     │
│  ┌─────┐  ┌─────┐  ┌─────┐         │
│  │存储器│  │转换器│  │广播塔│  ...   │
│  └─────┘  └─────┘  └─────┘         │
└─────────────────────────────────────┘
```

**重要**：本模组不提供核心方块！核心需要由其他兼容的 MOD 提供（如大型能源 MOD）。

#### 3. 基础设备搭建

**第一步：能量存储器**
- 用于存储多余能源
- 核心富余时自动充电
- 核心缺电时自动放电

**第二步：能量转换器**
- 右键切换转换模式
- SE ↔ GE ↔ FE ↔ AE
- 转换速率：1000 GE/s

**第三步：能源中继器**
- Shift+ 右键进入连接模式
- 右键目标完成连接
- 最大距离：500 格
- 黄色激光显示连接

**第四步：能源广播塔**
- 短距离无线传输（50 格）
- 自动扫描并连接设备
- 信号源位置：本体向上 3 格
- **注意**：需要与核心连接才能工作！

## 📦 方块介绍

### 标准能量存储器
- **容量**: 100,000 SE
- **用途**: 前期能源储备
- **特性**: 自动充放电，无需手动操作

### 通用能量存储器
- **容量**: 10²⁰ GE
- **用途**: 大型电网储能
- **特性**: 超高容量，支持巨型电网

### 能量转换器
- **功能**: 能源单位转换
- **模式**: 
  - SE → GE (1 SE = 10⁸ GE)
  - GE → FE (1 GE = 9×10¹⁸ FE)
  - GE → AE (需要应用能源 2)
- **操作**: 右键切换模式

### 能源中继传输器
- **连接距离**: 500 格
- **用途**: 长距离能源传输
- **操作**: 
  1. Shift+ 右键进入连接模式
  2. 对准目标右键
  3. 黄色激光表示连接成功

### 能源广播塔
- **传输距离**: 50 格
- **用途**: 短距离无线传输
- **特性**:
  - 自动连接范围内设备
  - 信号源在顶部 3 格
  - **必须连接核心才能工作**
- **视线要求**: 设备与信号源之间不能有≥3 层完整方块遮挡

### 能源标记方块
- **用途**: 多方块结构定位
- **特性**: 
  - 创造模式物品
  - 手持木棍可见
  - 无碰撞箱
- **操作**: 由其他 MOD 通过 API 放置和绑定

## 🌐 Web 监控界面

### 访问方式

1. 确保游戏已加载世界
2. 打开浏览器访问：`http://localhost:8080`
3. 查看实时电网状态！

### 界面功能

| 功能 | 说明 |
|------|------|
| **当前电量** | 核心存储的总能量 |
| **能量消耗** | 整个电网的耗电功率 |
| **能量产出** | 整个电网的发电功率 |
| **充满时间** | 预计核心充满所需时间 |
| **设备管理** | 查看并控制所有连接设备 |

### 设备管理 (v0.6.0+)

在 Web 界面底部可以：
- 查看所有已连接设备列表
- 查看设备位置和供能状态
- **点击开关** 控制设备供能

**提示**: 关闭不重要的设备可以节省能源！

## 🔧 高级玩法

### 电网规划建议

#### 小型电网 (新手)
```
核心 → 广播塔 → 设备
       ↓
     存储器 (备用)
```

#### 中型电网 (进阶)
```
        ┌─ 广播塔 ─ 设备群
核心 ───┼─ 广播塔 ─ 设备群
        └─ 中继器 ─── 广播塔 ─ 远程设备
              ↓
           存储器组
```

#### 大型电网 (专家)
```
                    ┌─ 区域 A ─┐
        中继器 ── 广播塔 ─ 设备
       /                     \
核心 ──┤          ┌─ 区域 B ─┐
       \         │
        └─ 中继器 ── 广播塔 ─ 设备
                     ↓
                  存储器阵列
```

### 广播塔布局技巧

1. **高度优势**: 将广播塔建在高处，扩大覆盖范围
2. **避免遮挡**: 信号源位置（塔顶 3 格）到设备之间避免完整方块
3. **中继器辅助**: 远距离设备使用中继器中转
4. **多层覆盖**: 大型基地使用多层广播塔网络

### 能源管理策略

- **峰谷平衡**: 用电高峰时使用存储器放电
- **储备充足**: 保持至少 1 小时产量的储备能源
- **分区供电**: 重要设备独立供电线路
- **监控预警**: 通过 Web 界面监控，低电量时报警

## ⚙️ 技术细节

### 核心唯一性系统

- 每个世界只能有一个能源核心
- 多 MOD 冲突处理规则：
  1. 优先选择等离子工作室提供的核心
  2. 没有则按加载顺序选择第一个
- 检测到多个核心会强制崩溃（防止数据损坏）

### 视线检查规则

广播塔传输需要视线畅通：

| 遮挡物 | 是否遮挡 |
|--------|---------|
| 空气 | ❌ 不遮挡 |
| 玻璃/树叶 | ❌ 不遮挡 |
| 半砖/楼梯 | ⚠️ 部分遮挡（<3 层可通过） |
| 完整方块 | ✅ 遮挡（≥3 层无法通过） |
| 液体 | ❌ 不遮挡 |
| 其他设备 | ❌ 不遮挡 |

**计算公式**: 
```
有效连接 = 设备任意方块未被完全遮挡 (≥3 层完整方块)
```

### 连接规则

**有效连接**:
```
核心 ──────→ 广播塔 ✓
核心 ──────→ 中继器 ──────→ 广播塔 ✓
核心 ──────→ 中继器 ──→ 中继器 ──→ 广播塔 ✓
```

**无效连接**:
```
广播塔 (未连接核心) ✗
广播塔 ──→ 中继器 (未连核心) ✗
中继器 ──→ 中继器 (未连核心) ✗
```

## 🎯 常见问题 (FAQ)

### Q: 为什么没有核心方块？
**A**: 本模组专注于能源管理，核心由其他大型能源 MOD 提供。这样可以更好地融入现有整合包。

### Q: 如何知道核心是否工作？
**A**: 访问 Web 界面 `http://localhost:8080`，如果显示能量数据说明核心正常工作。

### Q: 广播塔不工作怎么办？
**A**: 检查以下几点：
1. 是否与核心有直接或间接连接
2. 信号源位置（塔顶 3 格）到设备之间是否有遮挡
3. 距离是否在 50 格以内
4. 在 Web 界面查看设备是否在线

### Q: 存储器不充放电？
**A**: 这是正常的！存储器会自动工作：
- 核心能量富余时自动充电
- 核心能量不足时自动放电
- 无需手动操作

### Q: Web 界面无法访问？
**A**: 检查：
1. 游戏是否已加载世界
2. 端口 8080 是否被占用
3. 防火墙是否阻止
4. 查看游戏日志是否有错误

### Q: 如何移除设备连接？
**A**: 
1. 在 Web 界面找到对应设备
2. 点击供能开关关闭
3. 破坏设备即可自动断开连接

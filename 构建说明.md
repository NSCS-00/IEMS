3. 将 JAR 文件复制到 `.minecraft/mods/` 目录

## 常见问题

### Q1: "Unsupported class file major version" 错误

**原因**: Java 版本不正确

**解决**: 确保使用 Java 21

```cmd
java -version
```

### Q2: 插件下载失败

**错误**: `Plugin [id: 'net.neoforged.gradle.userdev', version: '7.0.145'] was not found`

**原因**: 无法访问 NeoForge Maven 仓库

**解决**: 
1. 配置 hosts 文件
2. 或使用代理服务器

### Q3: SSL 证书错误

**错误**: `PKIX path building failed`

**原因**: Java 无法验证 HTTPS 证书

**解决**: 
1. 更新 Java 的根证书
2. 或使用 HTTP 镜像

### Q4: 构建时间过长

**原因**: Gradle 下载依赖

**解决**: 
1. 使用 Gradle 缓存
2. 配置国内 Maven 镜像

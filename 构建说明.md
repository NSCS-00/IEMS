# IEMS 构建说明

## 环境要求

### 必需
- **JDK 版本**: Java 21 (推荐使用 Eclipse Temurin 或 Oracle JDK)
- **Gradle 版本**: 8.8 或更高版本 (但低于 9.0)
- **操作系统**: Windows 10/11, Linux, macOS

### 网络要求
构建过程需要访问以下 Maven 仓库：
- `https://maven.neoforged.net/releases` (NeoForge 插件)
- `https://maven.minecraftforge.net` (Forge 依赖)
- `https://repo.maven.apache.org/maven2` (Maven 中央仓库)

**注意**: 如果你在中国大陆地区，可能需要配置代理或使用镜像源。

## 构建步骤

### 1. 克隆仓库
```bash
git clone https://github.com/NSCS-00/IEMS.git
cd IEMS
```

### 2. 设置 JAVA_HOME
确保 `JAVA_HOME` 环境变量指向 Java 21 安装目录：

**Windows:**
```cmd
set JAVA_HOME=C:\Program Files\Java\jdk-21
```

**Linux/macOS:**
```bash
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
```

### 3. 构建
```bash
# 使用 Gradle Wrapper (推荐)
./gradlew build

# 或者使用系统 Gradle
gradle build
```

### 4. 查找输出
构建成功后，编译后的 JAR 文件位于：
```
build/libs/IEMS-0.6.0.jar
```

## 常见问题

### 1. "Unsupported class file major version" 错误
**原因**: Java 版本过高或过低  
**解决**: 确保使用 Java 21

```bash
java -version  # 检查 Java 版本
```

### 2. 插件下载失败
**错误信息**: `Plugin [id: 'net.neoforged.gradle.userdev', version: '7.0.96'] was not found`

**原因**: 无法访问 NeoForge Maven 仓库

**解决方案**:
1. 检查网络连接
2. 配置代理 (如果需要)
3. 使用镜像源 (修改 `settings.gradle`):

```gradle
pluginManagement {
    repositories {
        mavenLocal()
        gradlePluginPortal()
        maven { 
            url = 'https://maven.neoforged.net/releases'
            content {
                includeGroup("net.neoforged.gradle")
            }
        }
    }
}
```

### 3. SSL 证书错误
**错误信息**: `PKIX path building failed`

**原因**: Java 无法验证 HTTPS 证书

**解决方案**:
1. 更新 Java 的根证书
2. 或者暂时使用 HTTP (不推荐)

### 4. Gradle 版本不兼容
**原因**: 使用了 Gradle 9.0 或更高版本

**解决**: 降级到 Gradle 8.x

```bash
gradle --version  # 检查 Gradle 版本
```

## 开发环境设置

### IntelliJ IDEA
1. 打开项目
2. 等待 Gradle 导入完成
3. 运行 `runClient` 任务启动客户端

### VS Code
1. 安装 Java Extension Pack
2. 打开项目
3. 使用 Gradle for Java 扩展运行任务

## 运行客户端

```bash
# 使用 Gradle
./gradlew runClient

# 或使用 IntelliJ IDEA 的 Gradle 面板运行 runClient 任务
```

## 清理构建

```bash
./gradlew clean
```

## 发布构建

```bash
# 构建并发布到本地 Maven
./gradlew publishToMavenLocal
```
